# Anomaly Detection cloud resources management plane
## Introduce
This is a comprehensive visual management system designed for efficiently overseeing various cloud service platforms, including Kubernetes and OpenStack. It enables seamless control, management, and monitoring of cloud resources while incorporating an intelligent firewall that detects and mitigates abnormal external traffic.

### Dashboard
You can use dashbord instead of commands to control your cluster system like Kubernetes or OpenStack.
